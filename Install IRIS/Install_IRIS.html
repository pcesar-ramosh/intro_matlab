IRIS macroeconomic modeling and forecasting toolbox

By Cesar Ramos
The IRIS Toolbox is a free, open-source suite of tools for macroeconomic modeling and forecasting in MATLAB, developed by the IRIS Solutions Team since 2001.
It provides a user-friendly, command-oriented environment that integrates core modeling functions—such as a flexible model file language, tools for simulation, estimation, forecasting, and model diagnostics—with supporting infrastructure for data management, time series analysis, and reporting. (Iris)

Advantages of the IRIS Toolbox:

  • Comprehensive Modeling Environment: IRIS supports the entire workflow of macroeconomic modeling, from theoretical development to practical implementation.
  • Integration with MATLAB: Seamlessly integrates with MATLAB, leveraging its computational capabilities and user-friendly interface.
  • Open Source and Free: IRIS is open-source, allowing users to inspect, modify, and enhance its functionalities without licensing costs.
  • Extensive Documentation: Comes with detailed documentation and tutorials to assist users in effectively utilizing its features.

Installation Instructions:

Software Requirements:

  • MATLAB R2018a or newer is required for the stable branch of IRIS. The bleeding-edge branch requires MATLAB R2019b or later. For structural model estimation functions, the Optimization Toolbox is also needed. Iris Solutions Team

Download IRIS Toolbox:

  • Clone the repository using Git:
  • For the stable branch:
git clone --branch stable https://github.com/IRIS-Solutions-Team/IRIS-Toolbox.git /path/to/iris
  • For the bleeding-edge branch:
git clone --branch bleeding https://github.com/IRIS-Solutions-Team/IRIS-Toolbox.git /path/to/iris
  • Alternatively, download the ZIP file from the GitHub repository:
  • IRIS Toolbox on GitHub

Set Up IRIS in MATLAB:

  • Open MATLAB.
  • Navigate to the directory where you downloaded or cloned the IRIS Toolbox.
  • Add the IRIS Toolbox to your MATLAB path:
  • In MATLAB, go to the "Home" tab and click on "Set Path."
  • Click "Add with Subfolders," select the IRIS Toolbox directory, and click "Save."
  • Run the startup file to initialize IRIS:matlabCopiar códigoirisstartup

Verify Installation:

  • To confirm that IRIS is installed correctly, execute:matlabCopiar códigoirisping
  • If the installation is successful, MATLAB will display a confirmation message.

Monetary Policy Analysis and Forecasting (MOOC)

Learn about the macroeconomic motivation of the quarterly projection model (QPM), its key properties, model calibration, data filtration, and how to implement the QPM in MATLAB software in order to learn and understand practical model building and model operation as it is usually done in central banks.

The course covers two main technical aspects:
  • introduction to a canonical New Keynesian model structure and its key properties; and
  • implementation of the QPM in Matlab/Octave and the application of IRIS toolbox for solving and maintaining the QPM.
